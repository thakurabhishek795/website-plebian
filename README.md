# Plebian Collective Website

A modern, responsive website for Plebian Collective - a non-profit organization dedicated to societal betterment through employment, education, legal support, and environmental stewardship.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Built with Tailwind CSS for a clean, professional look
- **Interactive Gallery**: Filterable image gallery with lightbox viewer
- **Donation Page**: Dedicated page for accepting donations
- **SEO Optimized**: Proper meta tags and semantic HTML

## Pages

- `Main.html` - Homepage with mission, programs, and impact sections
- `Gallery.html` - Photo gallery with category filtering
- `Donation.html` - Donation form and information

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- JavaScript
- Font Awesome Icons
- Google Fonts (Inter & Merriweather)

## Setup

1. Clone this repository
2. Open `Main.html` in your browser
3. No build process required - all dependencies are loaded via CDN
