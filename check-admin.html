Check Admin Users

View existing admin users in Firebase Authentication

How this works:

  • This page checks if you're currently logged in to Firebase
  • If logged in, it displays your user information
  • To see all users, you need to check the Firebase Console

View All Users in Firebase Console:

  1. Go to Firebase Console
  2. Select your project
  3. Click "Authentication" → "Users" tab
  4. You'll see all registered users with their emails and UIDs
