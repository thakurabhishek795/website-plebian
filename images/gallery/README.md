# Gallery Images

Place your gallery images in this folder with the following naming convention:
- gallery-1.jpg (Community Outreach - Large featured image)
- gallery-2.jpg (Child Education)
- gallery-3.jpg (Tree Plantation)
- gallery-4.jpg (Women Empowerment - Wide image)
- gallery-5.jpg (Medical Camp)

Recommended image sizes:
- Large items: 1000x800px
- Standard items: 800x800px
- Wide items: 1000x400px
